# Heart of the Matter School of Music

This repository is being converted into a pnpm monorepo for the Heart of the Matter School of Music platform.

## Workspace

- `apps/web` - Next.js web app
- `packages/ui` - shared UI package placeholder
- `packages/config` - shared configuration package placeholder
- `packages/types` - shared TypeScript types package placeholder
- `docs` - project documentation
- `_archive_existing_repo_contents` - archived contents from the cloned legacy repository

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm lint
pnpm format
```
