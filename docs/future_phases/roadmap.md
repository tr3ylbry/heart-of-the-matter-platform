# Multi-Phase Roadmap

## Phase 1 — Private Studio MVP
Goal: Automate Liz's own student booking/payment/calendar workflow.

Features:
- Custom website
- Acuity booking
- Stripe payments
- Google Calendar sync
- Pricing/packages
- Contact and policies
- Current family migration

## Phase 2 — Optimization and Content Foundation
Goal: Improve conversion and prepare for scalable offerings.

Features:
- Testimonials
- Better SEO
- Blog foundation
- Songwriting masterclass waitlist
- Email capture
- Analytics

## Phase 3 — Workshops and Masterclasses
Goal: Create scalable education revenue.

Features:
- Paid workshops
- 10-week songwriting masterclass
- Replay access
- Worksheets/resources
- Cohort email flows

## Phase 4 — Teacher Pilot
Goal: Test adding educators without building a full platform.

Features:
- 1–3 teacher profiles
- Manual onboarding
- Booking pages
- Simple fee model
- Teacher agreement

## Phase 5 — Marketplace Foundation
Goal: Build actual teacher/studio platform only after pilot validation.

Features:
- Teacher accounts
- Teacher onboarding
- Stripe Connect
- Teacher payouts
- Teacher dashboard
- Student accounts
- Search/filter
- Reviews/testimonials

## Phase 6 — Courses, Community, Merch
Goal: Build the wider Heart of the Matter universe.

Features:
- Course library
- Community space
- Merch/print-on-demand
- Student showcases
- Guest lectures
- Educator resources

## Phase 7 — School Outreach / Curriculum
Goal: Turn Liz's school transformation experience into B2B offerings.

Features:
- School outreach page
- Case studies
- Curriculum packages
- Professional development workshops
- Consulting offers
