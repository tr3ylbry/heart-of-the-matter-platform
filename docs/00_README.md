# Heart of the Matter School of Music — Project Knowledge Base

This folder is a working project memory system for the Heart of the Matter School of Music website/platform.

Recommended workflow:
1. Keep this folder in the project repository under `/docs`.
2. Update `strategy/decision-log.md` whenever a decision is made.
3. Update `operations/open-questions.md` before and after every client call.
4. Treat `implementation/mvp-build-plan.md` as the current build target.
5. Move future ideas into `future_phases/roadmap.md` so they do not derail the first launch.

Current strategic recommendation:
Build a custom website for Heart of the Matter School of Music using Acuity Scheduling + Stripe + Google Calendar for the first version. Do not build the full teacher marketplace/payment system yet. Design the site so it can later expand into teacher profiles, workshops, courses, merch, school outreach, and community.
